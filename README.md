# Proyecto: Conceptos Fundamentales de Programación - Entrega 1

Este repositorio contiene el desarrollo de la **Entrega 1** para la asignatura de Conceptos Fundamentales de Programación. El sistema está diseñado en Java y se enfoca en la gestión, procesamiento y análisis de información comercial mediante archivos CSV.

## 🚀 Estructura del Proyecto
El sistema se divide en dos módulos principales:
1. **`GenerateInfoFiles.java`**: Herramienta encargada de crear los datos de prueba necesarios (vendedores, productos y ventas).
2. **`Main.java`**: Módulo de procesamiento que lee los datos, calcula totales y genera los reportes finales.

## 📊 Resultados y Pruebas
Se han incluido ejemplos de los archivos generados por el sistema (vendedores, productos y ventas) dentro de la carpeta **[DOC](https://github.com/ccvelez350-sys/cfpProyecto/tree/main/DOC)**. Estos archivos sirven como muestra del correcto funcionamiento de la lógica de programación aplicada.
